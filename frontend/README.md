# Фронтенд (Vite + React)

Фронтенд проекта ChatChat разработан как Single Page Application (SPA), написанное на React. Оно использует Vite в качестве быстрого инструмента для сборки.

## Описание структуры

*   **`src/`** — Основная папка с исходным кодом React-приложения.
    *   **`App.jsx`** — Главный и объемный компонент всего приложения. В нем сосредоточена инициализация интерфейса, вся стейт-машина (React State) управления пользователем, состоянием чата (выбранный чат, список сообщений, набор участников), а также логика по установке WebSocket-соединений.
    *   **`main.jsx`** — Точка входа. Файл, отвечающий за рендеринг `App` в "корень" HTML-файла страницы (`index.html`).
    *   **`styles.css`** — Глобальные стили для отображения интерфейса (отступы, цвета, оформление Chat UI).
*   **`package.json`** — Менеджер пакетов npm. Описывает нужные библиотеки приложения и различные npm-скрипты (например, для запуска: `npm run dev`).
*   **`vite.config.js`** — Инструкции и конфигурация сборщика Vite (настройки проксирования и портов для режима разработки).

## Взаимодействие с бэкендом
В данном приложении фронтенд берет на себя роль «клиента». Он отправляет данные на бэкенд двумя путями:
1.  **REST API запросы (HTTP)** — при помощи JS API `fetch`. Используются для разовых действий, таких как логин, регистрация, загрузка истории чата (из базы) или загрузка файлов на сервер;
2.  **Двустороннее соединение (WebSocket)** — после успешной авторизации `App.jsx` открывает сокет (встроенный API `WebSocket`), через который моментально принимает входящие сообщения от других пользователей, и отправляет собственные без задержек.
