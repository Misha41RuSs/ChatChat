# Chat Frontend

This frontend is built with Vite and React.

## Setup

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

2. Start the frontend server:
   ```bash
   npm run dev
   ```

3. Run the backend on `http://localhost:8080`.

The dev server proxies `/api` and `/ws/chat` to the backend to avoid CORS issues.
